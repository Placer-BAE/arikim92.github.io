<http://php.net/manual/kr/function.array-shift.php>

```php
<?php
// 첫번째 원소 삭제
$array = array( 'A', 'B', 'C', 'D', 'E' );
array_shift( $array );
print_r( $array );

> Array ( [0] => B [1] => C [2] => D [3] => E )
?>
```

<http://php.net/manual/kr/function.array-unshift.php>

```php
<?php
// 첫번째 원소 추가
$array = array( 'C', 'D', 'E' );
array_unshift( $array, 'A', 'B' );
print_r( $array );

> Array ( [0] => A [1] => B [2] => C [3] => D [4] => E )
?>
```
